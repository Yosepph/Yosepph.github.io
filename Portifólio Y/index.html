Alguns Conteito Front-end

  • GRACEFUL DEGRADATION
    Desenvolver um site pensando que nem todos tem acesso a um navegador atualizado que suporta CSS(estilização do site) e JavaScript(Linguagem de programação) bastante utilizada em criação de sites para criar funcionalidade em sites. Sendo assim, voltando o código em HTML semântico.
  • PROGRESSIVE ENHANCEMENT
    O Progressive Enhancement é uma abordagem que coloca ênfase na entrega de conteúdo funcional e acessível para todos os usuários, e então, adiciona camadas adicionais de melhoria para aqueles que podem aproveitar ao máximo as tecnologias mais avançadas, e ai que entra a responsividade.
  • AJAX
    "Asynchronous Javascript And Xml", é uma técnica de desenvolvimento web que permite que partes de uma página sejam atualizadas sem a necessidade de recarregar a página inteira. Isso é possivel através da combinação de várias técnologias incluindo, HTML, CSS, JAVASCRIPT e XML (embora o xml nem sempre seja utilizado atualmente, sendo frequentemente substituido por JSON = JavaScript Object Notation).
